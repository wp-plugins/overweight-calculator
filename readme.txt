== Description ==

This plugin displays functional overweight calculator / BMI calculator. It can be used to check the user's Body Mass Index and suggest the recommended weight range 

You can try the [overweight calculator demo](http://calendarscripts.info/overweight-calculator.html "Try Overweight Calculator Demo") to get an exact idea how this plugin will look over your blog.



== Installation ==

1. Upload overweight-calculator.php to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the CSS classes in the admin screen in Plugins/Overweight Calculator Configuration. This is optional, the calculator will work fine without any CSS classes.
4. Place `[overweight-calculator]` in the content of a page or post. The calculator will appear on the place of this tag
